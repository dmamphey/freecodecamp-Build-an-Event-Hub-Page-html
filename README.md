# Event Hub

A simple HTML project built as part of my coding practice.

## About the Project

This project is a basic event webpage called **Event Hub**. It features sections for upcoming and past events, with event descriptions, dates, navigation links, and images for past events.

The page was created to practise structuring content clearly using semantic HTML and organizing information into sections and articles.

## Features

- Main page heading
- Navigation menu with internal links
- Upcoming events section
- Past events section
- Event descriptions and dates
- Images for past events
- Simple and organized page structure

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using semantic HTML elements such as `header`, `nav`, `main`, `section`, and `article`
- Creating navigation links to sections within a page
- Organising event information clearly
- Adding images with alt text
- Structuring a webpage with multiple content sections
